Política de Privacidade

Esta é uma versão simplificada para fins acadêmicos.

1. Dados coletados

Podem ser coletadas informações como nome, e-mail e telefone informados no formulário.

2. Uso das informações

Os dados são utilizados apenas para contato relacionado aos serviços e cursos apresentados.

3. Armazenamento

Os dados podem ser armazenados em sistemas de terceiros utilizados apenas para demonstração.
